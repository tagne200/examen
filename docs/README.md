## 🛠️ Outils et Scripts

### Scripts Disponibles

| Script | Emplacement | Description |
|--------|-------------|-------------|
| `test-aws-credentials.bat` | `../scripts/` | Test credentials AWS |
| `deploy.bat` | `../` | Déploiement local |
| `validate-c21.bat` | `../` | Validation C21 |
| `check-prereqs.bat` | `../` | Vérification prérequis |

### Commandes Utiles

```bash
# CI/CD
gh run list                    # Lister les runs
gh secret list                 # Lister les secrets

# AWS
aws sts get-caller-identity    # Vérifier identité
aws s3 ls                      # Lister buckets

# Docker
docker-compose up -d           # Démarrer
docker-compose logs -f         # Logs

# Tests
pytest backend/tests/ -v       # Tests backend
```
